Everything you need. Nothing you don't.

A focused client with the features power users expect, built on a battle-tested engine.

libtorrent core

Runs on libtorrent-rasterbar 2.0 — the same engine behind qBittorrent and Deluge. Hundreds of simultaneous peers, full protocol support.

Full playback control

Pause, resume, and bulk-select torrents with checkboxes. Per-torrent download and upload speed limits. One-click open in Finder or Explorer.

Per-file priorities

Set individual file priorities inside a torrent. Skip what you don't need. Download the important files first without waiting for the full archive.

Automatic tracker injection

Pulls the best public trackers from ngosang/trackerslist every hour and adds them to every active torrent. More peers, better speeds, no manual setup.

UPnP port forwarding

Automatically opens and refreshes your router's port mapping so peers can connect inbound. No manual router configuration required.

Open source

Every line is readable, auditable, and forkable. No telemetry, no accounts, no analytics. Config is a single Python file you can edit directly.

Privacy controls that actually work.

Four independent protection layers, all configurable in the Settings dialog without touching a config file or the command line.

  • SOCKS5 / HTTP proxy Route all peer, tracker, and DHT traffic through your VPN or proxy. Optional username/password authentication.
  • Forced encryption Drops unencrypted peer connections. Resists ISP traffic shaping and DPI-based throttling.
  • Interface binding — kill switch Bind all traffic to a specific network interface (e.g. utun3). If your VPN disconnects, traffic stops automatically.
  • DHT and LSD disable Prevents your IP from being announced to the global DHT network and local service discovery.
